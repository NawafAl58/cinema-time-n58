# Cinema Time (سينما تايم) - Monorepo

Professional full-stack streaming platform built by **Nawaf (N58)**.

## Project Structure
- `client/`: Next.js frontend (Netflix-style UI)
- `server/`: Node.js Express API
- `prisma/`: Database schema and ORM
- `docker-compose.yml`: Local orchestration

## Quick Start
1. `npm install` (from root)
2. `docker-compose up`
3. Visit `http://localhost:3000`
